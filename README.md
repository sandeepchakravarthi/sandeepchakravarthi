      ### 🔍 SmartRAG Enterprise — Full-Stack RAG System
      [![Repo](https://img.shields.io/badge/View%20Repo-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/sandeepchakravarthi/smartrag)
      ![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)

      ![Qdrant](https://img.shields.io/badge/Qdrant-FF4F5A?style=flat-square&logoColor=white)

      > An enterprise-ready, production-grade Retrieval-Augmented Generation platform built from scratch — upload documents, ask questions, get precise AI-powered answers with source citations.
      >
      > | Layer | Technology | Purpose |
      > |---|---|---|
      > | **LLM (Local)** | Gemma 3 12B via Ollama | Free, open-source answer generation |
      > | **LLM (Cloud)** | GPT-4o / Gemini 1.5 Pro | Cloud fallback, highest quality |
      > | **Embeddings** | nomic-embed-text (768-dim) | Text → vector conversion |
      > | **Reranker** | cross-encoder MiniLM-L-6 | Precision top-K re-scoring |
      > | **Vector DB** | Qdrant | ANN search with filter support |
      > | **Metadata DB** | MongoDB | Document metadata & chat history |
      > | **Cache** | Redis | Query & embedding result caching |
      > | **Backend** | FastAPI + Uvicorn | Async REST API with JWT auth |
      > | **Frontend** | React + Vite | Real-time streaming UI via SSE |
      > | **DevOps** | Docker Compose | 6-service full-stack orchestration |
      >
      > **Pipeline:** `Query → Embed → ANN Search (top-20) → BM25 Hybrid → Rerank → Prompt → LLM → SSE Stream → MongoDB`
