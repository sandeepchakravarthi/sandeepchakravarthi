              ## 🌟 Featured Project

              ### 📋 [IntelliInvoice AI](https://github.com/sandeepchakravarthi/intelliinvoice)
              **AI-Powered Invoice Processing Platform**

              A full-stack application that automates invoice processing using intelligent agents:

              🔧 **Tech Stack:**
              - **Backend**: FastAPI (async Python), SQLAlchemy 2.0, PostgreSQL
              - - **Frontend**: React 18, TypeScript, TailwindCSS, Vite
                - - **AI Pipeline**: 5-agent workflow (OCR → Extraction → Validation → Fraud Detection → Approval)
                  - - **LLM**: Qwen 2.5:7b via Ollama
                    - - **OCR**: PaddleOCR + PyMuPDF
                      - - **Storage**: MinIO, Redis cache
                        - - **DevOps**: Docker, docker-compose
                         
                          - ✨ **Features:**
                          - - 📸 Invoice upload (PDF & images) with bulk processing
                            - - 🤖 AI-powered data extraction using LLMs
                              - - ✅ Intelligent validation with GSTIN & amount checks
                                - - 🚨 Fraud detection (duplicates, anomaly detection)
                                  - - 👥 Role-based approval workflows
                                    - - 📊 Real-time dashboard & analytics
                                      - - 🔐 JWT authentication, audit logging
                                        - - 🌐 Multi-user with role management (Admin, Finance User, Manager, Auditor)
                                         
                                          - **Status**: ✅ Production-Ready | 📈 Scalable Architecture | 🔒 Enterprise Security
