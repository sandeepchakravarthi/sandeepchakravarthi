## 🚀 Featured Projects

### 🧠 Smart Research Agent — LangGraph Agentic AI System
[![Repo](https://img.shields.io/badge/View%20Repo-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/sandeepchakravarthi/my-first-project/tree/langgragh)
![LangGraph](https://img.shields.io/badge/LangGraph-1.2-1C3C3C?style=flat-square&logo=langchain&logoColor=white)
![FastAPI](https://img.shields.io/badge/FastAPI-009688?style=flat-square&logo=fastapi&logoColor=white)
![Ollama](https://img.shields.io/badge/Ollama-000000?style=flat-square&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2496ED?style=flat-square&logo=docker&logoColor=white)

> A production-grade **autonomous AI agent** that thinks, plans, and uses tools to answer queries — built with LangGraph's agentic loop architecture.

**What the agent can do:**

| Tool | Capability |
|---|---|
| 🔍 **Web Search** | Real-time internet search via SearXNG (open-source, free) |
| 📄 **PDF Reader** | Extract and analyze uploaded PDF documents |
| 📋 **Summarizer** | Condense long documents into key insights |
| 🧮 **Calculator** | Safe mathematical computation via AST eval |
| 🎤 **Voice Input** | Speech-to-text via Web Speech API |
| 🔊 **Voice Output** | Text-to-speech responses in browser |
| 🧠 **Memory** | Full session-based conversation context |

**Agentic Flow:**
```
User Query → LangGraph Agent Loop → Tool Selection → Tool Execution → LLM Reasoning → Final Answer
```

**Multi-LLM Support:** Ollama (Qwen3/Llama3 — local, free) · GPT-4o (OpenAI) · Claude (Anthropic)

### 🔍 SmartRAG Enterprise — Full-Stack RAG System
[![Repo](https://img.shields.io/badge/View%20Repo-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/sandeepchakravarthi/smartrag)
![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![FastAPI](https://img.shields.io/badge/FastAPI-009688?style=flat-square&logo=fastapi&logoColor=white)
![React](https://img.shields.io/badge/React-61DAFB?style=flat-square&logo=react&logoColor=black)
![Qdrant](https://img.shields.io/badge/Qdrant-FF4F5A?style=flat-square&logoColor=white)
![MongoDB](https://img.shields.io/badge/MongoDB-47A248?style=flat-square&logo=mongodb&logoColor=white)

> An enterprise-ready **Retrieval-Augmented Generation** platform — upload documents, ask questions, get AI-powered answers with cited sources. Powered by hybrid search, cross-encoder reranking, and multi-LLM support.

**RAG Pipeline:**
```
Query → Embed (nomic-embed-text) → ANN Search top-20 (Qdrant) → BM25 Hybrid → Rerank (cross-encoder) → LLM → SSE Stream
```

| Layer | Technology | Purpose |
|---|---|---|
| **LLM** | Gemma 3 12B / GPT-4o / Gemini 1.5 Pro | Answer generation |
| **Embeddings** | nomic-embed-text 768-dim | Text → vector |
| **Vector DB** | Qdrant | ANN search |
| **Reranker** | MiniLM cross-encoder | Precision top-K |
| **Cache** | Redis | Query caching |
| **Metadata** | MongoDB | Docs, chat, users |
| **Backend** | FastAPI + JWT auth | Async REST + SSE |
| **Frontend** | React + Vite | Real-time chat UI |
| **DevOps** | Docker Compose (6 services) | One-command deploy |
